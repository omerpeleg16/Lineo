# LINEO Brand Guidelines

## Brand Personality

LINEO is a modern AI product for instructional designers. The interface should communicate clarity, trust, professionalism and creativity while remaining minimal and calm.

The visual language should feel closer to a premium SaaS product than to an academic project.

### Core Values

- **Simple**
- **Intelligent**
- **Clean**
- **Calm**
- **Precise**
- **Modern**

*Avoid playful, colorful or overly futuristic styles.*

---

## Visual Style

The interface uses a soft, minimal design language with generous white space, rounded corners and subtle shadows.

The overall feeling should be light, elegant and effortless.

### Recommended Elements:
- Large white surfaces
- Soft borders
- Subtle depth
- Rounded components (16–24px radius)
- Smooth animations
- Spacious layouts

### Elements to Avoid:
- Strong shadows
- Saturated colors
- Sharp corners
- Heavy gradients
- Glassmorphism
- Neumorphism

---

## Color Palette

### Primary
- **Deep Teal (`#075056`)**
- **Used for:** Primary buttons, Titles, Active states, Progress indicators, Icons, Logo

### Secondary
- **Light Mint (`#E4EEF0`)**
- **Used for:** Background elements, Selected states, Cards, Inputs

### Neutral
- **Soft Blue Gray (`#F1F7F8`)**
- **Used for:** Secondary backgrounds, Panels, Large content areas

### White
- **Soft White (`#FBFBFB`)**
- **Used for:** Cards, Surfaces, Main content

### Background Strategy
The application does not use a flat background. Instead, it uses a very subtle radial gradient that creates depth without distracting from the interface.

**Recommended Background Style:**
- **Base color:** `#F1F7F8`
- Large radial glow behind the main content
- Slight transition toward white in the center
- Soft teal tint near the edges
- Very low opacity
- No visible color bands

The background should create the feeling that the interface is gently illuminated from behind. It should never compete with the content.

---

## Typography

- **Primary Font:** Noto Sans Hebrew

### Hierarchy:
- **H1:** 32px — Bold
- **H2:** 24px — SemiBold
- **Body:** 18px — Regular

*Use generous spacing between sections. Avoid long paragraphs.*

---

## Components

### Buttons
Large rounded buttons.
- **Primary:** Deep teal background (`#075056`), White text
- **Secondary:** White background, Teal border (`#075056`)
- **Hover:** Slight elevation and darker teal.

### Cards
- White surface (`#FBFBFB`)
- 16–20px border radius
- 1px light border
- Very soft shadow
- Large internal padding

### Icons
- Outline style
- Rounded
- Single color (Primary Teal `#075056`)

### Progress Indicator
The progress indicator is a key visual element of LINEO.
- Vertical timeline
- Connected stages
- Active teal state
- Completed stages with checkmarks
- Subtle inactive states

*Progress should communicate clarity and confidence.*

### Illustrations
Generated assets should always appear:
- Centered
- Evenly spaced
- On clean backgrounds
- With generous whitespace

*Never overcrowd the screen.*

---

## Motion & Animations

Animations should feel **smooth**, **purposeful**, and **calm**.
- **Recommended duration:** 200–400ms

### Avoid:
- Bouncing
- Exaggerated motion
- Flashy transitions

---

## Overall Experience

The interface should feel like a premium AI creation tool.

The user should experience:
`Upload → Configure → Generate → Receive`
with minimal cognitive load.

Every design decision should reduce complexity while emphasizing the quality and consistency of the generated graphic kit.
